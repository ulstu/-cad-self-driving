Ваша задача: сделать так, чтобы робот проехал 20 см по прямой и остановился. Функция setPosition(x) поворачивает колесо на x радианов. В упражнении используется робот e-puck. Радиус колеса этого робота 20 мм. На вкладке Инструкции подробно объясняется как редактировать исходный код контроллера

Программирование роботов на языке Python

  1. Откройте редактор исходного кода: кликните на роботе правой кнопкой мыши и выберите пункт "Edit controller". После этого откроется исходный код контроллера.
  2. Отредактируйте контроллер: на 9 строке измените значение переменной position
  3. Сохраните программу: Нажмите на кнопку сохранения или воспользуйтесь комбинацией клавиш Ctrl+S.
  4. Перезапустите симуляцию
  5. Проверьте ваш результат: После перезапуска симуляции робот будет двигаться прямо. Ваш результат будет обновляться в реальном времени на вкладке Метрики. Вы можете в любой момент отредактировать контроллер и перезапустить симуляцию, чтобы достичь лучшего результата.
